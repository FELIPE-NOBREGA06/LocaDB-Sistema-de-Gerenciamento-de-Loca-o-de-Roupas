## Como Adicionar GIFs

1. Grave a tela do seu sistema usando ferramentas como:
   - OBS Studio (gratuito)
   - ScreenFlow (macOS)
   - Camtasia
   - ShareX

2. Converta para GIF usando:
   - FFmpeg: `ffmpeg -i video.mp4 -vf "fps=10,scale=1280:-1:flags=lanczos" output.gif`
   - Online: https://ezgif.com/video-to-gif

3. Salve os GIFs nesta pasta com os nomes sugeridos acima

4. Atualize o README.md principal com as referências aos GIFs
